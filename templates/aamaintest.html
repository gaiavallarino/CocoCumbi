About Coco Cumbi
Coco Cumbi allows data visualization and manipulation from tree census data to support the user in decision-making policy progress when it comes to (re)forestation.

This version of the software, personalized for the Pichincha province Public Administration uses data coming from the canton Ruminahui tree census dataset, which deals with some typically endangered tree species, such as the Araucaria pine, the Chilean Palm and our homonym, the Coco Cumbi
